# StudentInterviewHub

A web platform for college students to practice interviews, book mock interview slots, and stay updated on internships, placements, and hackathons. Built with **MERN stack** and **AI features using Gemini API**.

---

## Features

- **Executive Club:** Apply and manage interview slots  
- **Slot Booking:** Book online/offline mock interviews  
- **Opportunity Feed:** Latest internships, hackathons, placements  
- **AI Features:** Mock interview generator, resume analyzer  

---
